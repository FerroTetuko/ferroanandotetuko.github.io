Lulusan SMK Muhammadiyah Kudus dengan jurusan atau peminatan Teknik Komputer Jaringan. Memiliki Pemahaman dasar yang kuat dalam sistem operasi Windows, pengelolaan jaringan komputer, protokol jaringan SSH, FTP, TCP, DHCP, HTTP, HTTPS, mengkonfigurasi perangkat router Mikrotik, simulasi topologi jaringan. Instalasi sistem operasi Debian 12 pada mesin virtual, mengelola jaringan dan server Linux. Terbiasa menggunakan software seperti Microsoft Word, Excel, Power Point untuk mengolah data, Winbox untuk Mikrotik, Virtual Box dan VM Ware untuk virtualisasi operasi sistem, FileZilla untuk FTP, Bitvise untuk SSH.
